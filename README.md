* Prequisites
  * python3.8
    - black

* How to run the game ?

  .. code:: bash
     python3.8 game.py

  and follow the onscreen instructions

* How to run the test ?

  .. code:: bash
     python3.8 -m unittest test_game.py

* How to format the code ?

  .. code:: bash
     python3.8 -m black -l 110 game.py test_game.py
